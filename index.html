Common "find" usage scenarios

1) Search multiple directories

pwd> find dir1 dir2 -name "*.txt"

2) This gives the full path of the files that you are searching for.

find dir1 dir2 dir3 \( -name "*.xxx" -o -name "*.yyy" \)  -exec ls -lrt {} \; | grep -i somename
